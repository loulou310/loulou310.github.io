Page Github

Ce projet est né du conseil de mon professeur de NSI : déposer un lien vers mon compte Github sur ParcoursSup pour ceux qui regardent les dossiers. A la place, j'ai décidé de créer ce site hébergé sur Github, en utilisant le HTML et apprendre a utiliser la technologie bootstrap

Le but de ce site est donc de présenter brièvement les projet personnels en programation.

Le projet initial était d'utiliser un CMS que j'ai déployé sur un hébergeur. Mais j'ai trouvé plus valorisant dans un dossier de faire le site internet moi-même

Ce site est donc construit grace a Bootstrap et hébergé sur Github grace au système Github pages.
